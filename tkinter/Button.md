A widget that, when clicked, performs an action.
# Inicialize the variable
```python
button = Button(root, text="Click me!",
    background='red',         # The background color of the button
    font=('Arial', 20),       # The font to use for the button's text
    foreground='white',       # The text color of the button
    borderwidth=5,            # The width of the 3D border around the button
    relief='solid',           # The style of the 3D border
    activebackground='green',  # The background color of the button when it is active
    activeforeground='white'  # The text color of the button when it is active
)
```
